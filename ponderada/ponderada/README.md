# Projeto Ponderada - MVC + Supabase

## Como rodar

1. Instale as dependências:
```
npm install
```

2. Rode o servidor:
```
npm start
```

3. Acesse no navegador:
```
http://localhost:3000
```

## Funcionalidades

- Leitura da tabela `compromissos` no Supabase
- Exibição de tarefas com HTML + JS (JSON)
